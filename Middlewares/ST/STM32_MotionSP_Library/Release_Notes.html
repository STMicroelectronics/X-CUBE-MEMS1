Release Notes for MotionSP Signal Processing Library

Update History

V2.2.0 / 30-May-2019

Main Changes

  • Added new APIs to get some variables related to MotionSP so the following ones will be deprecated
    (AcceleroODR, AccMagResults, MotionSP_Parameters, TimeDomainData)

V2.1.0 / 09-May-2019

Main Changes

  • Added new API for Time Domain and Frequency Domain Analysis

V2.0.0 / 08-February-2019

Main Changes

  • New data acquisition method based on accelerometer FIFO in continuous mode
  • Added subrange for thresholds analysis
  • Added Overlapping
  • Added saving the Max FFT Number evaluated
  • Modified some API for optimizing and restyle the code

V1.0.0 / 01-September-2018

Main Changes

  • First version of the library published under the ODE initiative
